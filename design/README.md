2. Study the layout, colors, fonts, and spacing
3. Try to reproduce it in the corresponding lab folder

## Design Specification

All labs use the following design tokens:

### Colors
| Name | HEX | Usage |
|------|-----|-------|
| Background | `#F8FAFC` | Page background |
| Text | `#0F172A` | Headings, dark text |
| Muted | `#64748B` | Paragraphs, secondary text |
| Primary | `#2563EB` | Buttons, links |
| Secondary | `#0EA5E9` | Secondary buttons |
| White | `#FFFFFF` | Cards, modal background |
| Border | `#E2E8F0` | Card borders, dividers |
